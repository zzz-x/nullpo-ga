# HCI-Project

### 人机交互大作业

#### 运行项目：

```sh
# backend
# ./backend/:
pip install -r requirements.txt
flask run
```

```sh
# frontend
# ./web/:
yarn install
yarn run start
```

#### 贡献：

​		贡献=贡献数量（行数决定）x贡献质量（最终投票决定）

​		按照为项目添加的核心行数(非自动生成，非引入)来计算贡献数量

- 流程/项目框架 贡献数量=行数x1.3
- 代码 贡献数量=行数
- 报告/ppt/ppt汇报 贡献数量=行数x0.8（只是初步，之后再按照实际情况讨论决定）
